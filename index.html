WHAT IS GravityX?

GravityX is a Top-down shooting game demonstrating Software Engineering project. The game is

doesnâ€™t have the fanciest graphic, but it has real intense gameplay and innovative features. GravityX

gives you the chance of experience the real gravity in space ship while not losing the fun element of top-

down game. You will control a spaceship with or without a partner and roam in the infinite space facing

enormous enemies. But be aware, there is a catch, not only the space ship that you are controlling will

fall into the gravity of the planets, but also the bullet you shoot out of the ship will be catch by the

gravity, not mentioning that the planet can become your biggest enemy too. You will try to avoid the

planets face towards you as well as the enemy ships. You will be challenged to destroy the final boss

with different kind of weapon and health upgrade. And you will be dealing with a lot of physics.

Installation

The java application contain a single jar file written with JavaFX and JVM is required to run this program.

Gameplay

The game will launch with a window containing main menu. The window is in fixed resolution

and recommend screen resolution is above 1280*800. This game feature a single player or multiplayer

game play option. The game also feature sprite elements, animation and gameplay music. The game

also feature score system that log and record each kill and lose. The player ship will be destroyed if the

health drop below 0 .

Control

The user can control the ship movement with UP, DOWN, LEFT or RIGHT keys in keyboard and shoot

bullets with SPACE key. The user can also return to the main menu by Escape key.

Space Ship

The player will control a single type space ship. The enemy has different class of spaceships. Each

spaceship class will feature different attributes, which include speed, size and different icon. And each

enemy ship have different behavior.

Upgrades

Random type of upgrades will appear during the game. And the emerge location is also non-

deterministic.

Enemy Waves

During the game play as time progress on, many waves of enemy spaceship will emerge from the right

side of the space. For each wave of enemy ships, the number and type of enemy ship occurs differently

as well as spawn location.

Story Mode

The game presents a story. When the game is over, the user will watch the end story by

characters of the player and professor Bhola.
